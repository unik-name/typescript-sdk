# UNS Typescript SDK

## How to use

Please visit [SDK page](https://docs.uns.network/uns-usethenetwork/sdk.html) from UNS documentation.

## How to contribute

### Run tests

`yarn test`

### Build project

`yarn build`

### Release

On develop branch:
`yarn release`

_NB: use semver_
