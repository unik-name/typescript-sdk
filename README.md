# UNS Typescript SDK

## How to use

Please visit [SDK page](https://docs.uns.network/uns-usethenetwork/sdk.html) from UNS documentation.

## How to contribute

### Run tests

`yarn test`

### Build project

`yarn build`

### Bundle for production

`yarn bundle`

### Release

`npm publish`

_NB: use semver_
